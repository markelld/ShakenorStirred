# Project Overview

## Project Name

Shaken or Stirred ?  

# https://shorst.netlify.app

## Project Description

This site consists of a catalog of classic coctails and Foxbar creations in two catagories Shaken or Stirred. The site has the functionality to get,delete and post cocktail recipes.

#### MVP 

- Use AirTable API for cocktail data.
- Allow user to create, delete and search recipe. 
- Create contact page

#### PostMVP  

- Add authentication feature which will allow user or establishment to save there own recipes to there account.
- Enable user to add photo to new cocktail recipe submission.
